# Place your logo here as `logo.png`

Flutter will load it via:
  Image.asset('assets/images/logo.png')

The splash screen, auth screen, and onboarding all display this logo
in a 72×72 rounded square. Recommended spec:
  - Size: 512×512 px (Flutter downscales automatically)
  - Format: PNG with transparency
  - Style: white icon/symbol on a transparent background
            (the green gradient container behind it is added in code)
